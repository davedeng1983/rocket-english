# 错误归因准确性优化策略

## 当前问题
- AI 分析可能不够准确
- 没有充分利用题目的元数据（解析、知识点等）
- 没有分析学生错误答案和正确答案的差异
- 缺少题目类型特定的分析策略

## 优化策略

### 1. 利用题目元数据
- **题目解析（analysis）**：从解析中提取关键知识点和错误原因
- **知识点标签（meta.kps）**：优先使用题目已有的知识点
- **题目类型（section_type）**：根据题型提供特定分析

### 2. 错误答案对比分析
- **错误答案 vs 正确答案**：分析为什么学生会选错
- **选项干扰项分析**：分析错误选项的干扰点
- **常见错误模式**：识别典型的错误模式

### 3. 多维度信息融合
- **上下文信息**：结合题目、选项、文章、解析等所有信息
- **优先级策略**：优先使用高置信度的信息源
- **降级策略**：AI → 解析提取 → 知识点标签 → 规则生成

### 4. AI 提示词优化
- **提供更多上下文**：包括错误答案、正确答案、解析、知识点
- **要求分析"为什么选错"**：基于错误答案反推原因
- **多轮分析**：先识别题型，再分析具体原因

### 5. 规则增强
- **基于错误答案的规则**：如果选了B，分析B为什么错
- **选项差异分析**：分析正确选项和错误选项的差异
- **题型特定规则**：完形填空、阅读理解等特定规则
